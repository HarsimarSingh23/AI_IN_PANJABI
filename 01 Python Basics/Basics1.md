- Variables and Data Types
- Operators

Variables - 
In Python, a variable is a container for storing data values. You can create a variable by assigning a value to it using the equals sign (=). For example:

```python
x = 5
y = "Hello, World!"
```     

Why we need this ? 
because the computer has memory and we need to store data in that memory. 
Variables allow us to do that. Variable internally maps to a memory location where data is stored. 
We can use variables to store numbers, strings, lists, and other types of data.
Data Types -
Python has several built-in data types that we can use to store different kinds of data. Some of the most common data types in Python include:
- int: for storing integers (whole numbers)
- float: for storing floating-point numbers (decimal numbers)
- str: for storing strings (text)
- bool: for storing boolean values (True or False)
- list: for storing a collection of items (ordered and mutable)
- tuple: for storing a collection of items (ordered and immutable)
- dict: for storing key-value pairs (unordered and mutable)
- set: for storing a collection of unique items (unordered and mutable)

Example - 
```python
x = 10          # int
y = 3.14        # float
name = "Harman"  # str
